# 🎨 WB Fashion Card Generator (Groq + Pollinations.ai + Pillow)

![Streamlit](https://img.shields.io/badge/streamlit-1.30+-FF4B4B.svg)
![Pillow](https://img.shields.io/badge/Pillow-10.1+-green.svg)
![License](https://img.shields.io/badge/license-MIT-black.svg)

Интеллектуальный ассистент для селлеров Wildberries. Приложение генерирует креативные идеи для принтов на футболках, создает качественные изображения с помощью ИИ, автоматически накладывает их на реалистичный мокап одежды, добавляет настраиваемую инфографику и позволяет в один клик выгрузить готовую карточку в личный кабинет поставщика WB.

---

## ✨ Ключевые возможности

*   **🤖 Автогенерация идей и SEO (Groq API):** Генерация продающих названий, описаний с ключевыми словами (оверсайз, хлопок, подарок) и готовых промптов для ИИ на основе анализа трендов.
*   **🎨 Генерация принтов (Pollinations.ai / Flux):** Создание уникальных графических принтов высокого разрешения без водяных знаков.
*   **👕 Реалистичный мокап (Pillow):** Автоматическое наложение сгенерированного принта на базовую модель футболки с имитацией текстуры ткани и прозрачности.
*   **🏷️ Динамическая инфографика:** Быстрое создание привлекательных плашек (акценты на состав, крой, бренд) с возможностью выбора цвета и текста прямо в интерфейсе.
*   **☁️ S3 Облачное хранилище:** Автоматическая загрузка финальных карточек в облако (AWS, VK Cloud, Yandex Cloud и др.) для генерации постоянных ссылок.
*   **📤 Интеграция с Wildberries API:** Создание номенклатуры товара и автоматическая привязка медиафайлов напрямую в ваш кабинет продавца.

---

## 🏗️ Архитектура проекта (SOLID / DRY / KISS)

Код спроектирован по принципам чистого кода и разделен на независимые слои:

*   `app.py` — Тонкий слой представления (Streamlit UI). Отвечает только за рендеринг интерфейса и работу с сессией.
*   `config.py` — Безопасное чтение конфигураций и переменных окружения, централизованное хранение констант.
*   `services/ai_service.py` — Работа с нейросетями (текстовая модель Groq Llama 3 и генератор изображений Flux).
*   `services/image_service.py` — Локальная обработка изображений, наложение слоев, работа со шрифтами и отрисовка плашек инфографики.
*   `services/s3_service.py` — Интеграция с S3-совместимыми хранилищами для публикации медиафайлов.
*   `services/wb_service.py` — Взаимодействие с официальным API Wildberries.

## 🚀 Быстрый старт

### 📋 Требования
*   Установленный **Python 3.10+**
*   Файл шаблона футболки **`tshirt_base.png`** (с прозрачным фоном) в корневой папке проекта.

### 1. Клонирование и установка зависимостей
```bash
git clone [https://github.com/ваш-логин/имя-репозитория.git](https://github.com/ваш-логин/имя-репозитория.git)
cd имя-репозитория
Создайте виртуальное окружение и установите библиотеки:

Bash
python -m venv venv

# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

pip install -r requirements.txt
2. Настройка переменных окружения
Создайте файл .env в корневой директории проекта и добавьте ваши ключи:

Ini, TOML
# API ключ Groq (можно получить бесплатно на console.groq.com)
GROQ_API_KEY=gsk_your_groq_api_key_here

# Настройки S3 (например, VK Cloud, Yandex Cloud или AWS)
S3_ACCESS_KEY_ID=your_access_key
S3_SECRET_ACCESS_KEY=your_secret_key
S3_ENDPOINT_URL=[https://hb.bizmrg.com](https://hb.bizmrg.com) # Или ваш эндпоинт
S3_BUCKET_NAME=your_bucket_name
3. Запуск приложения
Bash
streamlit run app.py
После запуска интерфейс откроется в вашем браузере по адресу http://localhost:8501.

🛠️ Как пользоваться генератором
Шаг 1. Генерация идеи: Нажмите кнопку 🤖 Придумать идею и SEO-текст. Groq сгенерирует название футболки, SEO-описание и промпт для принта. Вы можете отредактировать любой текст вручную.

Шаг 2. Настройка дизайна: Разверните вкладку 🎨 Настройка инфографики для настройки плашек. Измените текст (например: "100% ХЛОПОК") и выберите цвета.

Шаг 3. Рендеринг: Нажмите 🚀 Создать принт и карточку. Система создаст принт, перенесет его на футболку и добавит стильные плашки. Полученный результат можно сразу скачать кнопкой 📥 Скачать готовую карточку.

Шаг 4. Экспорт на WB: Введите ваш токен партнера Wildberries (с правами на раздел "Контент"), укажите артикул и нажмите 📤 Создать карточку и загрузить на WB. Карточка мгновенно появится на маркетплейсе.

🛡️ Безопасность (Security First)
Конфиденциальность ключей: Все чувствительные данные (ключи API, S3-доступы, токены) вынесены в файл .env и добавлены в .gitignore. Они никогда не попадут в публичный доступ на GitHub.

Прямая выгрузка: Код не хранит ваши WB-токены. Они передаются по безопасным HTTPS-протоколам напрямую на сервера Wildberries.
